# Contribution Rules (WIP)

Basic rules and guidelines for how to contribute to this package.

## 1. All Effects Should be Fully Text Based

The purpose of this package is to use character by character manipulation to create cool effects. No fancy CSS based effects, and actual character by character styling should be used sparingly. TLDR: if you couldn't recreate the effect in a CLI it probably doesn't fit the vibe of this package.

## 2. No Extra Dependencies

This package only has a couple dependencies: react, ts, tsup. I would like to keep this as lightweight as possible, and anything you'd be adding to this package almost certainly doesn't NEED new deps. Its just string manipulation, not rocket science.
